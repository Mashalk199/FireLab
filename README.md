How to run app:

1. Get the Firebase plist file from either Mashal or Yihan and place it in the root directory.
2. Create an account on twelvedata.com
3. Create a new plist file called "Secrets.plist". 
4. Create a key-value pair and name the key "FinancialAPIKey" and put your twelvedata API key as the value of the pair.
5. Cmd + R on Xcode.
